Filter Method

The filter method is great, if you would like to find something in an array based on a condition. Simply add .filter to the end of the array and you can use a simple call back function to stipulate the condition you want to use when filtering. The filter method returns back all in the array that meets the condition, an array is returned no just the value.

You can pass three paramters into the filter method, below is each paramter with a short explination

  • *Required* - Value - This is each value in the array that you are filtering on
  • *Optional* - Index - The array index of the current element
  • *Optional* - arr - The array object the current element belongs to

Reduce Method

The reduce method is great, if you would like to find the total of something in an array. Simply add .reduce to the end of the array and you will pass in the accumulator and the current value as paramters. The accumulator could start by being 0 and you can set this value right after the call back function. Eventually creating a final sum which would be the returned accumulator.

You can pass four paramters into the filter method, below is each paramter with a short explination

  • *Optional* - Accumulator - The accumulator is the value that we end with. We start with a value and build on it to get a final value
  • *Required* - Value - This is each value in the array that you are filtering on
  • *Optional* - Index - The array index of the current element
  • *Optional* - arr - The array object the current element belongs to
